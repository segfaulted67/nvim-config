** My Neovim Configuration

  This is my personal Neovim configuration. It uses [[https://github.com/folke/lazy.nvim][lazy.nvim]] for plugin management.


** Installation

  1. Clone this repository:
    #+BEGIN_SRC bash
    # for codeberg
    git clone https://codeberg.org/segfaulted67/nvim-config.git ~/.config/
    mv nvim-config nvim/
    # for github
    git clone https://github.com/segfaulted67/nvim-config.git ~/.config/
    mv nvim-config nvim/

    #+END_SRC
  2. Open Neovim and run `:Lazy sync`
